# 操作符

一个最简单的表达式的例子:2+3。这个表达式可以被拆解为操作符和操作数。以下是一些常用的操作符。

|   |   |   |   |
|---|---|---|---|
|操作符|名称|功能|示例|
|+|加|返回两数之和|a=3;b=4;a+b返回7；|
|-|减|返回两数之差|a=3;b=4;a-b返回-1;|
|*|乘|返回两数之积，效果同power()函数|a=3;b=4;a*b返回12；|
|**|幂运算|x**y,求x的y次幂|x=3;b=4;a**b返回81；|
|/|除|返回两数之商的浮点数(python3)|a=6;b=3;a/b返回2.0；|
|//|地板除|返回两数之商向下取整的整数|a=8;b=3;a//b返回2；|
|%|模运算|即取余(对正数而言)|a=8;b=5;a%b返回3；|
|<<|左移|二进制下左移，最低位补0|2<<2返回8；|
|>>|右移|二进制下右移，最低位舍弃，最高位补符号位0|11>>1返回5；|
|&|按位与|二进制下，2个数各个位置做与运算|5&3返回1；|
|\||按位或|二进制下，2个数各个位置做或运算|5\|3返回7；|
|^|按位异或|二进制下做异或运算(有且仅有一个为真即为真)|5^3返回6；|
|~|按位取反|x按位取反得 -(x+1)|~5返回-6|
|<|小于|比大小，返回布尔值|5<3返回False；|
|>|大于|比大小，返回布尔值|5>3返回True;|
|<=|小于等于|比大小返回布尔值|3<=6返回True;|
|>=|大于等于|比大小返回布尔值|3>=6返回False；|
|\==|值相等|判断字面值常量是否相等|3\==2返回False；'str'\=='Str'返回False；|
|!=|值不相等|判断字面值常量是否不等|'str'!='Str'返回True；|
|not|布尔否|布尔取反|x=True;则not x返回False|
|and|布尔与|若|x=False;y=True;x and y返回False|
|or|布尔或||x=True;y=False;x or y返回True|

## 注意事项

### 除运算

python3以后，即使两数能恰好整除，也会返回商的浮点数。python2则只返回整型；

### 地板除运算

地板除总是返回一个小于商本身的整数。例如：8除以3得商2.666666无限循环，与该商接近的整数有2和3。因此8//3=2

### 位运算

```python
2<<2==8          # 2的二进制为0010，左移2位即1000,即十进制下的8
11>>1==5         # 11的二进制为1011，右移1位即0101,即十进制下的5
5&3==1           # 即0101 & 0011,得0001，即十进制下的1
5|3==7           # 即0101 | 0011,得0111，即十进制下的7
5^3==6           # 即0101 ^ 0011,得0110，即十进制下的6
```

### 布尔短路

python中的布尔与、或运算遵循短路原则。在了解短路原则之前，我们先明确一个python默认的规则：

在python中：**None、任何数值类型中的0、空字符串“”、空元组()、空列表[]、空字典{}**都被当作False，还有自定义类型，如果实现了 　__ nonzero __ ()　或　__ len __ () 方法且方法返回 0 或False，则其实例也被当作False，其他对象均为True。

我理解的python的短路原则可以总结为：

在一次布尔运算中，如果已经出现可以最终确定该次运算的结果的表达式，则本次布尔运算直接返回该表达式的值**，并且不再评估另一个参与运算的表达式的结果。**

这句话的解析如下：

**逻辑与**：在逻辑运算中，与运算遵循**一假即假，全真才真**。与作为二元运算符，与运算的一般格式为 x and y。

- 当x为**假**时，本次与运算结果就确定为假，运算直接返回x表达式的结果，而y表达式不再被求值；
- 当x为**真**时，并不能直接确定本次与运算的结果，所以另一个参与运算的y不会被忽略，并且成为了决定性因素，因此本次运算将返回y表达式的值。

```python

def a():
	print("a")
	return 0
	
def b():
	print("b")
	return 1
	
def c():
	print("c")
	return 1
	
def d():
	print("d")
	return 1

"""a()返回0,直接决定了a() and b()的运算结果，所以b()被忽略,a()继续参与后续运算;
    同理,c()和d()也被忽略
"""
result = a() and b() and c() and d() 
print("最终返回值：",result)
#输出结果为：
a
最终返回值：0
```

---

**逻辑或**：在逻辑运算中，或运算遵循**一真即真，全假才假。**或运算的一般格式为 x or y。

- 当x为**真**时，本次或运算的结果就确定为真，运算直接返回x表达式的结果，而y表达式不再被求值；
- 当x为**假**时，y的表达式结果就成了或运算的决定性因素，因此y不会被忽略，本次运算返回y的值；

```python
def a():
	print("a")
	return 0
	
def b():
	print("b")
	return 1
	
def c():
	print("c")
	return 2
	
def d():
	print("d")
	return 3
"""
a()函数返回0,不足以确定或运算结果，继续看b()函数的返回值，所以用b()的返回值继续参与运算；
b() or c()中，c()被忽略；b() or d()中，d()被忽略。
所以最终只有a()和b()函数会运行。
"""
result = a() or b() or c() or d()
print("最终返回值：",result)
#输出结果为：
a
b
最终返回值：1
```

---

**混合运算**：混合运算可以拆解成局部的与运算、或运算，同样遵循以上原则。

```python
def a():
	print("a")
	return 0
	
def b():
	print("b")
	return 1
	
def c():
	print("c")
	return 2
	
def d():
	print("d")
	return 3

def e():
	print("e")
	return 0

def f():
	print("f")
	return 4

def g():
	print("g")
	return 0

def h():
	print("h")
	return 5
	
result = a() and b() and c() or d() or e() and g() or h()
print("最终返回值：",result)
#输出结果为：
a
d
最终返回值： 3
```

# 运算优先级

python中的运算优先级同数学中一样：优先级高的先运算；相同优先级的自左向右结合；通过圆括号可以改变运算优先级。以下是各操作符按优先级**从低到高**的排列情况：

|   |   |
|---|---|
|操作符|描述|
|lambda|lambda表达式|
|or|布尔或|
|and|布尔与|
|not|布尔否|
|in,not in|成员判断，例如元素是否在序列中|
|is, is not|身份判断，例如判断两个对象的id是否相同|
|<, <=, >, >=, !=, \==|比较操作符|
|\||按位或|
|^|按位异或|
|&|按位与|
|<<,>>|移位符|
|+, -|加减|
|*, /, //, %|乘，除，地板除，模运算|
|+x, -x|正负|
|∼x|按位否|
|**|幂运算|
|x.attribute|属性引用|
|x[index]|按索引检索|
|x[index1:index2]|切片|
|f(arguments ...)|函数调用|
|(expressions, ...)|绑定操作或元组展示|
|[expressions, ...]|列表展示|
|key:datum, ...|字典展示|

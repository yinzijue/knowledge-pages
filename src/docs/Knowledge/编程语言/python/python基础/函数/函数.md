# 概述

函数是一段可重用的程序片段，它的本质是对功能的封装。你可以给这个程序片段一个名称，之后你就可以在程序的任何地方、任意次数地执行这个片段，使用函数名称的这样一过程，我们称之为调用函数。

# 定义与调用

定义一个函数的格式形如：

```python
def func_name(args):
    """docString"""
    funcBody
    return value

func_name(values)
```

- def：定义函数的内置关键字；
- funcName：函数名，遵循python命名规则
>[!info]-
>![[001关于python中的命名规范]]
- args：函数的形式参数列表；
- docString:文档字符串，如果存在，通常是函数中的第一条语句，用于简要说明函数功能。
- funcBody：函数主体，通常是一系列的操作步骤；

函数就像一个加工厂，传递不同的参数(提供不同原料)。可以得到相应的结果(对应的产出)。

- return value:函数返回值。返回值是可选的，当在调用有返回值的函数时，可用一个变量接收返回值。
>[!tip]+
>- 实际上，即使没有写明返回值，该函数也是有返回值的，不带return返回语句的函数返回None；
>- return执行后，函数会终止，return语句之后的语句将不会执行；
>- 若一个函数想要返回多个值，可以借助复合数据类型(list,tuple,set等)来操作；

# 关于函数

## 局部变量与全局变量

### 局部变量

当一个变量声明在函数内部时，它们与函数外具有相同名称的其他变量没有任何关系，即变量名称对于函数来说是局部的。我们称这些变量为这个函数的局部变量。这个局部变量可被访问的范围我们称之为这个局部变量的作用域。

提示建议在理解局部变量、全局语句、非局部语句前，先了解关于**作用域**方面的知识。
>[!info]-
>![[005关于python中的命名空间和作用域]]
```python
# x是func函数中的局部变量
def func():
    x = 5
# func函数外的方法无法访问函数内部的局部变量  
print(x)
```

### 全局变量

当你试图为一个函数外的变量赋值时，你就需要告诉函数这个变量不是局部的。在python中我们使用global关键字来实现这一操作。下面这个例子，你可以通过注释掉global所在行来感受下它的作用。

```python
t = 6


def say_hello():
    global t
    t = 3
    return t


say_hello()
print(t)
```

### 非局部变量

上面给出了如何在局部和全局作用域内使用变量。在python中还有一种作用域叫做**非局部作用域**，它不属于以上两种范围中的任何一种，你通常会在函数嵌套的场景中看到它。要使用它，需要用**nonlocal**关键字来告诉内部函数这是一个非局部变量。

```python
# 外部函数
def outer():
    x = 1
    print("x is",x)
    # 内部函数
    def inner():
        nonlocal x
        x = 2
    inner()
    print("now x is",x)
outer()
```

>[!tip]+ 
>你同样可以通过注释掉关于非局部变量的说明，来体会它的作用。

## 函数注解

函数注解是可选的用户自定义函数类型的元数据(metadata)信息。函数注解可以分别对函数的形参和返回值进行标注。

**形参标注**：定义格式为在形参名后加冒号，后面跟一个表达式，该表达式的求值结果会作为形参标注的值；

**返回值标注：**定义格式为一个组合符号**->**，符号后面跟一个表达式，该内容位于形参列表和表示def语句结束的冒号之间；

```python
def func(ham:str,egg:str='egg')->str:
    print("Annotations:",func.__annotations__)
    
 
>>>func('spam')
    
```

## 匿名函数

同其他编程语言一样，python中也有匿名函数。

- 所谓匿名函数，即不再使用def这样的标准形式来显式定义函数，这样产生的函数不具备函数名称，也无法在其他地方被调用；
- python中使用lambda关键字来创建匿名函数；
- lambda函数的语法形如：

```python
lambda [arg1[,arg2,...argn]]:expression

# eg.
sum = lambda arg1,arg2:arg1+arg2
```

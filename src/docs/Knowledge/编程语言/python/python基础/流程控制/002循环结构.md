循环结构常用于处理重复任务。python中的循环结构有两种：while和for。

# while循环

```python
while condition1:
    while condition2:
        statements
    
```

- while循环支持嵌套。只要条件(condition)满足，while循环会一直循环下去。没有跳出循环的条件的循环被称为死循环。

# for循环

```python
for t_var1 in iterable1:
    for t_var2 in iterable2:
        statements
```

>[!tip]+
> - python中的for语句与C或Pascal中的不同，它常与in联合用于迭代序列，例如列表、字符串等。在这些序列的迭代过程中，元素的迭代顺序与系列中出现的顺序一致。
> -  t_var：临时变量，通常为一些有意义的名词或简写；iterable：可迭代对象，通常为一些容器类数据，例如字符串、列表、元组、字典等；


# 其他循环相关

## range()函数

range()是python的一个内置函数，可以生成算数级数，常在循环结构中用于遍历**数字**序列。

```python
for var in range(begin,end,step):
    statements
```

- begin:起始项；end：最终项；step：步进(支持负数，表示倒序)。其中，步进是可选的，若从0开始迭代，起始项也可以省略；
>[!important]+
>- range()函数返回的序列不包含end值，即一个左闭右开区间。
>- 当range()函数与len()函数组合使用时，可以按索引来迭代序列，当然，通常我们使用enumerate()函数来更快地实现这一效果。


```python
a=[1,2,3]
for i in range(len(a)):
    print(a,a[i])
```
>[!info]+
>range()函数在迭代时，返回的对象基于所需序列返回连续项，并没有生成真正的列表，从而节省了大量空间；

## break语句

break语句可以用在循环结构中用于跳出当前循环，它适用于for和while。通过编写一些小demo，你可以感受到它的效果。

```python
# break跳出最近的循环
for i in range(10000):
    if i == 5:
        break
else:
    print(i)
```
>[!attention]-
>注意这里的else是从属于for语句而非if语句，因而该循环没有输出结果，直接跳出循环(0~4不符合if，5直接跳出for循环)。若修改else语句的缩进使之从属于if语句，则有输出。
## continue语句

continue语句用于在循环中跳过当前轮次的循环，直接开始下一轮次的循环。

```python
# continue跳出当前轮次的循环
for i in range(10000):
    if i == 5:
        continue
    else:
        print(i)
```

## pass语句

- pass语句本身不执行任何操作，但出于语法的完整性，必须要一个语句，此时可以使用pass语句。
>[!tip]-
>pass语句在实际开发中，常用于函数、类、条件子句的占位符，表示当前还未想好具体的逻辑，先临时占位。程序会直接忽略pass，之后，你可以重新回来完成这一部分。

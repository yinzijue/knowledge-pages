# 单项分支

```python
if condition:
    statements1
```

- condition：分支结构的条件表达式，该表达式为真，则执行statement1；若为假则不执行；

# 双项分支

```python
if condition1：
    statements1
else condition2:
    statements2
```

condition1结果为真时，执行statements1；结果为假时执行statements2；

# 多项分支

```python
if condition1:
    statements1
elif condition2:
    statements2
elif condition3:
    statements3
else:
    statements4
```

- elif分支可以无限添加，但只会执行其中一个，执行完一个分支后分支结构结束，其他分支不会判断也不会执行；
- 在python中没有switch-case语句，如果想实现匹配效果，可以用if-elif分支语句实现。当然，
>[!tip]+
>在python3.10之后，引入了match-case语句，相比if-elif而言，可以实现更强大的功能，具体可参阅官方文档[PEP 636-match-case](https://peps.python.org/pep-0636/ "关于match-case")

```python
def http_error(status):
    match status:
        case 400:
            return 'Bad request'
        case 401:
            return 'Unauthorized'
        case 403:
            return 'Forbidden'
        case 404:
            return 'Not found'
        case _:
            return 'Unknown status code'
	
```

# 巢状分支

巢状分支简单来说就是其他分支结构的嵌套模式，所有分支都可以嵌套；

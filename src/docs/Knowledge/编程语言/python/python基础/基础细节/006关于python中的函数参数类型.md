# 关于python中的函数参数类型
## 概述

初学python的时候，看到官方教程里关于函数参数的各种名词：形参、实参、位置参数、关键字参数、仅限关键字参数...有点迷惑，又急于搞清楚它们的区别及使用场景。随着学习的深入，私以为从函数的定义和调用两个过程来分别理解，可以将问题分解成各个简单的部分。

### 形参和实参

- 定义时：在定义函数时，在参数列表中声明的变量即函数的**形参**(形式参数)；
- 调用时：在调用函数时，我们传递给参数的值就是**实参**(实际参数)**；**

提示参数传递是一个值传递的过程；

```python
# 定义函数sayHello,形参为stmt
def sayhello(stmt):
    print(stmt)
    
# 调用sayHello函数，传入一个字符串    
sayhello("hello,world!")
 
```

## 默认值参数

### 定义

简单来说，在**函数定义**的形参名后加上赋值运算符(=)和默认值，这个参数就是函数的默认值参数，赋予的值就是默认参数值。

### 用法

- 定义时：在定义函数时，默认值参数带有默认值。在参数列表中，默认参数必须位于不带默认值的参数之后；
- 调用时：在调用函数时，默认参数的传参是可选的，你可以为它传递参数(使用默认值)，也可以传递新的参数。

提示在调用函数时，多个默认参数值之间的顺序没有限制，但和定义函数时一样，传参顺序也遵循“**先非默认，再默认”**。

```python
# times有默认值
def say(message,times=1):
    print(message*times)
 
# 未传值到times
say("hello")
# 将其他实参传递给times,也可以写成(“hello”,times=5)
say("hello",5)
```

## 关键字参数

### 定义

在**定义函数**时，默认参数值的形式类似于kwarg=value。在调用函数时，也有这样的形式，我们称之为**关键字参数**。

### 用法

- 定义时：关键字参数不涉及定义(事实上，在定义时不带默认值的被称为**位置参数**)
- 调用时：在函数调用时，关键字参数的传值形式为kwarg=value(kwarg是必选的)。

```python
def func(a,b,c=3):
    print(a,b,c)
   

func(3,)
```

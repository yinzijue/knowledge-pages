# 为什么使用变量

字面值常量足以存储信息。但实际应用中，我们需要一种既可以储存信息又可以对这些量进行**操作**的方法。基于此，我们引入了变量——它们的值可以变化，因此可以用于存储任何东西。为了便于访问这些变量，我们会给这些变量命名。

# python变量命名规则

- 变量名只能以**字母**或者下划线_开头；
- 变量名不能包含空格；
- 变量名不能与python关键重合，但可以包含关键字(python命令行下，可以输入help('keywords')查看当前版本支持的关键字)
- 变量名中不能有特殊字符，例如#@？等
- 变量名对大小写敏感；

基于以上规则，下面是一些合法的变量名和非法的变量名：

```python
# 非法的变量名
2things  *a  ?things               # 未以字母或下划线开头
student id                         # 变量名中包含了空格
and False def                      # 使用了python关键字充当变量名
a?head                             # 变量名中包含了特殊字符
##########################################################################
# 合法的变量
student_id
CUSTOM_NAME
__my_name
you_and_me
i
name_23
name22
```

- 在这6种常用的数据类型中：

- 不可变数据类型：**Number、String、Tuple**；
- 可变数据类型：**List、Dictionary、Set**；

### Number(数字)

python3中支持四种类型的数字：int(整型)、float(浮点型)、bool(布尔型)、complex(复数型)

- int：在python3中只有一种整型，即int，默认为长整型，例如35、24、1045等。提示python3中不再支持long；
- float：浮点型，可以简单理解为小数，形如：2.5、-.02等。12E-5这样的科学计数法也是浮点型；
- bool:布尔型，python3中**True**和**False**成为了关键字，其中True的值为1，False的值为0，这意味着你可以对指向布尔型对象的变量进行数值运算；
- complex：复数型，和数学中的复数相同，形如：4+3j等；

### String(字符串)

提示在python中没有单独的char类型，这意味着在python中，一个单独的字符就是一个长度为1的字符串；

- 在python中凡是用成对引号包含的内容都是字符串(单引号、双引号、三引号)，形如：**'a'** **"abc"** **"""_a12@"""**等。

#### 操作字符串

- 字符串支持通过加号(+)直接连接，也支持通过星号(*)对当前字符串进行复制，例如：

```python
a = "hello"
b = "world"
c = "python" * 3
print(a+b)
print(c)
```

- 字符串支持切片截取操作，你可以通过切片获取到字符串的一部分，例如：

```python
a = "hello"
print(a[3:])
print(a[1:3])
print(a[:3])
print(a[::3])
```

重要切片的范围是一个**左闭右开**的区间，使用时应注意取舍；切片的起始下标、结束下标和步进都是可选的；

- 此外，字符串还支持一些常用的操作函数，例如**len()**获取长度、**replace()**替换局部、**index()**获取索引、**find()**内容校验、**split()**拆分字符串等。

```python
a = 'hello,world'
print(len(a))                         # 返回字符串a的长度
print(a.replace('o', 'a'))			  # 将字符串a中的字母o替换为字母a
print(a.index('w'))					  # 返回字符串中字母w第一次出现时的索引
print(a.find('l', 0, 1))			  # 在切片中查找字母l第一次出现时的索引,-1表示未找到
print(a.split(',', 2))				  # 以逗号为分割符，对字符串a进行最多2次分割，返回一个列表
```

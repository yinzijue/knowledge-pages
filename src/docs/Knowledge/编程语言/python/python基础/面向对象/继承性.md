# 继承

面向对象编程带来的主要好处之一就是代码的重用，实现这种重用的方法之一就是通过继承机制。继承的基本语法如下：

```python
# DerivedClassName继承自BaseClassName
class DerivedClassName(BaseClassName):
    <statement-1>
    .
    .
	.
    <statement-2>
```

- BaseClassName:当前定义的类的**基类**，一般也称为其父类；
- DerivedClassName：派生自BaseClassName的**派生类**，一般也称为其子类；
- 当请求一个类属性时，如果在派生类中找不到，则按基类继承链逐步向下查找；
- 派生类可以重写其基类的方法；

python提供了两种内置函数可被用于继承机制：

- isinstance(obj,class)：用于检查一个实例的类型。当且仅当 ojb.\__class__为class或class的派生类时，isinstance(obj,class)返回True；
- issubclass(sub_class,base_class):用于检查类的继承关系。当且仅当sub_class是base_class的一个派生类时，issubclass(sub_class,base_class)返回True；

```python
>>>isinstance(1,int)           # True
>>>isinstance(1.0,int)		   # False
>>>isinstance(1.0,float)       # True
##################################################
>>>issubclass(bool,int)        # True
>>>issubclass(float,int)       # False
```

# 多重继承

python也支持多重继承，即一个派生类继承自多个不同的基类。多重继承的基本语法如下：

```python
class DerivedClassName(Base1,Base2,Base3):
    <statement-1>
    .
    .
	.
    <statement-N>
```

# markdown

## 在markdown标记与正文之间至少保留一个空格，使之生效

## 基本
- 标题
    - \#加一个空格，1个#代表一个标题级别
- 段落
    - 段落与段落之间用空白行分隔
- 换行
    - 使用HTML中的<br>实现
- 强调
    - *代表斜体；\**代表加粗；\***代表加粗斜体
- 块引用
    - \>将一个段落放到引用块中；>>表示引用嵌套
- 列表
    - 列表内嵌套其他元素：将其他元素缩进四个空格
    - 有序：数字+英文句号；无序：-短横
- 代码
    - 一个反引号是代码；2个反引号是代码内反引号转移；3个反引号可以指定代码类型，在第一个反引号后跟代码类型，可以实现语法高亮
- 分隔线
    - 单独一行  ≥3个短横实现分隔线效果(前后均加上空白行)
- 链接
    - 通用

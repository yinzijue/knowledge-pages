# 文件与目录

## 一切皆文件

- 不同与windows系统，在linux系统中，所有内容都是以文件的内容形式保存(包括硬件)，并且不依靠扩展名来区分文件类型。
- linux有如下三种常见的文件类型：

|   |   |
|---|---|
|普通文件|如文本文件、shell脚本、Java源码等，可使用vi、cat、more、less来进行管理；|
|目录文件|就是目录，相当于windows中的文件夹，包括目录名、子目录名，可使用ls列出目录的内容；|
|链接文件|类似于windows下面的快捷方式；|

## linux目录

- linux的文件系统采用的是层级式的树状目录结构，根目录“/”是所有目录的顶级目录；
- 以根目录为开头的路径称为绝对路径，反之，则称为相对路径；
- linux常用的系统文件目录如下：

|   |   |
|---|---|
|/|文件系统的顶级目录，一般称为根目录；|
|root|存放root用户的相关文件，root是linux系统下的超级用户，拥有全部权限；|
|home|系统预设的普通用户的家目录，存放普通用户的相关文件；|
|bin|(binary)存放所有用户都可以使用的**常用命令**的目录；|
|sbin|系统管理相关的二进制文件存放目录，要具有一定权限才可以使用其中的命令；|
|etc|存放各种配置文件|
|var|存放经常变化的文件，如网络连接的socket文件；|
|boot|启动目录，存放引导系统启动的相关文件；|
|usr|(Unix System Resource)安装软件的默认目录，类比windows下的program files|
|dev|设备文件保存目录，例如cpu、disk等；|
|lib|系统库保存目录；|
|mnt|(mount)默认挂载光驱和软驱的目录；|
|opt|(optional)额外安装的应用程序所在目录，例如用户自行下载的程序即可安装在此；|
|proc|只存在与内存中而不占用额外存储的目录，其中存储了当前系统的所有信息，包括进程、文件系统、硬件等；|

# 常用快捷键

|   |   |
|---|---|
|tab|自动补全|
|ctrl+c|(Cancel)强制终止当前命令的执行，|
|ctrl+l|清屏，相当于命令clear；|
|ctrl+z|把命令放到后台；|
|ctrl+r|在历史命令中搜索；|
|ctrl+a|(ahead)光标移动到命令行首|
|ctrl+e|(end)光标移动到命令行尾|
|ctrl+u|清除剪切光标到行首之间的内容(不含光标所在位置)；|
|ctrl+k|清除剪切光标及光标之后的所有内容(含光标所在位置)；|
|ctrl+y|在光标处粘贴剪切的内容；|
|ctrl+b|左移一位，同键盘左方向键；|
|ctrl+f|右移一位，同键盘右方向键；|

# 文件与目录管理

## 目录管理

- ls（英文全拼：list files）: 列出目录及文件名
- cd（英文全拼：change directory）：切换目录
- pwd（英文全拼：print work directory）：显示目前的目录
- mkdir（英文全拼：make directory）：创建一个新的目录
- rmdir（英文全拼：remove directory）：删除一个空的目录
- cp（英文全拼：copy file）: 复制文件或目录
- rm（英文全拼：remove）: 删除文件或目录
- mv（英文全拼：move file）: 移动文件与目录，或修改文件与目录的名称

## 文件管理

- cat 由第一行开始显示文件内容
- tac 从最后一行开始显示，可以看出 tac 是 cat 的倒着写！
- nl 显示的时候，顺道输出行号！
- more 一页一页的显示文件内容
- less 与 more 类似，但是比 more 更好的是，他可以往前翻页！
- head 只看头几行
- tail 只看尾巴几行

## 磁盘管理

- **df**（英文全称：disk free）：列出文件系统的整体磁盘使用量
- **du**（英文全称：disk used）：检查磁盘空间使用量
- **fdisk**：用于磁盘分区
- **mount:**磁盘挂载与卸载
- **fsck(**file system check**):**检查和维护不一致的文件系统

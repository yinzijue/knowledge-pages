# 软件测试

## 测试阶段
- 冒烟测试
- 单元测试
- 集成测试
- 系统测试
    - 功能、性能、可靠性、安全性
- 回归测试
- 验收测试
    - α测试
        - 内测
            - 用户在开发环境下进行的测试/非测/开人员在模拟实际环境下进行测试
- β测试
    - 公测
        - 由最终用户在一个或多个场所进行测试
- γ测试
	- 公测

## 是否覆盖源码
- 黑盒测试(数据驱动)
	- 功能测试
		- 逻辑功能测试
		- 界面测试
		- 易用性测试
		- 安装测试
		- 兼容性测试
	- 非功能测试
		- 性能测试
			- 负载测试
			- 压力测试
		- 安全性测试
- 白盒测试(逻辑驱动)
	- 语句覆盖
	- 判断/分支覆盖
	- 条件覆盖
	- 判断-条件覆盖
	- 条件组合覆盖
	- 路径测试
	- 灰盒测试

## 是否运行
- 静态测试
- 动态测试

## 测试手段
- 人工测试
- 自动化测试

# 安装

1. 安装 Python（3.9 或更高版本）
2. 安装软件包（如果安装失败，请检查维基百科）

```python
$ pip3 install locust
```

3. 验证安装是否成功

```python
$ locust -V
locust 2.27.0 from /usr/local/lib/python3.10/site-packages/locust (python 3.10.6)
```

4. 完成！现在你可以创建你的第一个测试程序

# 预发布版本

如果你需要 Locust 的最新版本并且不能等待下一个正式发布版本，你可以像这样安装一个开发构建版本：

```python
$ pip3 install -U --pre locust
```

每次有分支/PR合并到主分支时，都会发布预发布版本。
